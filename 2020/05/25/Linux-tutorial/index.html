初识Linux系统

开机与关机

  • 开机启动的程序，在Windows中叫做服务(Server)，在Linux系统中叫做守护进程(daemon)

  • 三种用户登录的方式：

    • 命令行登录：输入用户名与密码
    • 远程登录：云服务器ssh (Xshell)
    • 图形界面登录：Ubuntu

  • 最高权限账户为root账户

  • Linux中没有输出结果即代表输入的命令成功

  • 服务器用在Linux上基本不用到关机命令。在服务器中如果要关机需要先进行数据同步

 

系统的目录结构

  • 一切皆文件
  • 根目录/，所有的文件都挂载在这个节点下

Linux系统的目录结构：

  • /bin：Binary的缩写，存放常用的Linux命令（勿动）
  • /boot：Linux启动的核心文件，包括一些连接文件以及镜像文件（勿动）
  • /dev：Device设备的缩写，存放系统的外部设备（U盘），与mnt目录功能类似
  • /mnt：用户临时挂载文件系统目录，如挂载光驱
  • ==/etc：用来存放所有的配置文件，如变成环境。很重要！==
  • /home：用户主目录！类似于Win中的Administrator用户。一般在home目录下，不同的用户创建不同的文件夹来存储个人文件
  • /lib：存放系统最基本的动态链接共享库，类似于win的dll文件（勿动）
  • /lost+found：存放突然关机的文件
  • /media：挂载系统自动识别的设备如U盘或者光驱
  • ==/opt：安装的软件存放位置==
  • /root：管理员用户目录，即超级权限用户目录
  • ==/usr：很多应用程序和文件存放处，类似于windows中的program files==
  • /tmp：临时文件目录，用完即丢的文件
  • ==/var：经常修改的文件，比如日志文件==
  • ==/www：存放服务器网站相关的资源，环境，网站项目等==

 

常用的基本命令

目录管理

ls命令：列出当前目录下所有文件的命令

参数说明：

  • -a：all，查看全部的文件，包括隐藏文件
  • -l：列出所有的文件，包括文件的属性以及权限，看不到隐藏文件
  • -al：所有的参数可以组合使用，-al可以查看所有文件（包括隐藏文件）以及他们的属性和权限

 

cd命令：切换目录

  • 绝对路径：路径的全称。C:\ProgramData\Microsoft Office。==绝对路径：linux中以/开头，代表从根目录开始的绝对路径==
  • 相对路径：以当前所在路径为基准的路径。比如现在处在ProgramData目录下面，则下级目录相对于当前目录就是/Microsoft Office。==相对路径是对于当前目录该如何去寻找==

 

pwd：显示用户当前所在的目录

 

mkdir：创建一个目录

 

rmdir 删除目录

 

cp：复制文件或者目录

 

rm：移除文件或者目录。危险命令

  • -f：忽略不存在的文件，不会出现警告，即强制删除
  • -r：递归删除目录
  • -i：互动，删除的时候询问是否删除

 

mv：移动文件或者目录；两个文件执行重命名操作，一个文件执行移动操作

  • -f：强制
  • -u：只替换已经更新过的文件

 

基本属性

一个类型字母+3x3=9个读写权限的字母，三组代表三种用户

首先了解文件的属性：ls -l

观察首字母：常用的三种

  • [d]：代表目录
  • [-]：代表的是文件，-后面一般接的是文件的读写权限
  • [l]：代表链接文档

后面的九个固定字母：三个一组

  • [r]代表可读，[w]代表可写，[x]代表可执行

  • 三组从前到后分别为root用户所属权限、属组用户权限以及一般用户权限，中间的-代表没有该项操作的权限

  • 后面两个root，分别代表属主和属组（用户组）

 

==chomd==：更改文件的9个属性（常用）

文件属性设置有数字和符号两种，常用数字。比如用户有owner/group/others三种身份，每一种身份都有read/write/execute三种文件执行权限，每个权限对应的数字为

每种身份的举例：

  • owner = rwx = 7
  • group = rwx = 7
  • others = — = 0

 

文件内容查看

  • ==cat==：从第一行开始查看文件，读文章或者读取配置文件
  • tac：从最后一行开始查看文件
  • nl：显示的时候顺便输出行号，看代码**
  • more：一页一页的显示文件内容，空格代表翻页，回车代表向下翻行，观看的时候输入:f代表查看当前的行号
  • less：与more类似，
    • 可以往前翻页（空格键翻页，上下方向键可以翻页）
    • 使用/char指定字符串搜索向下查询
    • 使用?char指定字符串搜索向上查询
    • 要查询字符串，n代表向下查询，N代表向上查询
    • 使用q键离开文件
  • head：查看头几行
  • tail：查看尾几行

硬链接与软链接(了解)

Linux硬链接：

  • 硬链接：A–B，假设B是A的硬链接，他们两个指向同一个文件，允许一个文件拥有多个路径，用户可以通过这种机制建立硬链接到重要文件上防止误删
  • 软链接：类似win中的快捷方式，删除了源文件快捷方式也访问不了

例子：

touch：创建文件

echo：将字符串输入文件，或者从文件中输出字符串

上述情况，当删除了f1，软链接f3无法访问，而硬链接f2仍然能够访问
